A selection of engineering and biomedical projects

CANX-2 Satellite Link Simulator

RF Communications MATLAB | SDR | Signal Processing

Developed a comprehensive MATLAB-based satellite communication simulator for the CANX-2 mission. The simulator models complex RF propagation phenomena including Doppler shift effects, path loss calculations, satellite tumbling attenuation, and includes hardware playback capabilities for Software Defined Radio (SDR) testing.

Key Features:

  • Dynamic Doppler frequency modeling
  • Path loss and atmospheric attenuation calculations
  • Tumbling satellite attitude effects
  • Real-time SDR hardware integration

3D Ultrasound Motor Control System

Biomedical Imaging C++ | Qt | Embedded Systems

Designed and implemented embedded control software for a robotic 3D ultrasound imaging system. Developed a Qt-based graphical user interface for real-time motor control, probe positioning, and image acquisition, enabling precision medical imaging applications.

Key Features:

  • Real-time motor control and feedback
  • Intuitive Qt GUI for probe manipulation
  • Robotic 3D scan coordination
  • Image acquisition and processing pipeline

Optical Coherence Elastography

Biomedical Imaging OCT | Wave Propagation | MATLAB

Investigated mechanical wave propagation in biological tissue using OCT-based imaging techniques. This research explores elastography methods for tissue characterization and disease detection through non-invasive optical imaging.

Key Features:

  • OCT signal acquisition and processing
  • Mechanical wave propagation analysis
  • Tissue elasticity characterization
  • Advanced signal processing algorithms

Pulse Oximeter

Biomedical Instrumentation Signal Processing | Embedded C | Hardware Design

Developed a non-invasive pulse oximetry device for real-time blood oxygen saturation (SpO₂) measurement. The system utilizes red and infrared LED sensors with advanced signal processing algorithms to extract physiological parameters from photoplethysmographic signals.

Key Features:

  • Dual-wavelength LED sensing (Red & IR)
  • Real-time SpO₂ calculation
  • Heart rate detection
  • Signal processing for noise rejection
  • Compact wearable design

Electrocardiogram (ECG) Monitor

Biomedical Instrumentation Signal Processing | Analog Electronics | Python

Designed a multi-channel ECG acquisition and analysis system for cardiac signal monitoring. Implemented signal conditioning, filtering, and real-time analysis algorithms for heart rate and arrhythmia detection with a user-friendly interface.

Key Features:

  • Multi-channel ECG acquisition

  • Heart rate variability (HRV) analysis
  • Arrhythmia detection algorithms
  • Real-time visualization and logging
