Pulse Oximeter

Biomedical Instrumentation | Signal Processing | Embedded C

Overview

Developed a non-invasive pulse oximetry device for real-time blood oxygen saturation (SpO₂) measurement. The system utilizes red and infrared LED sensors with advanced signal processing algorithms to extract physiological parameters from photoplethysmographic signals.

Key Features

  • Dual-wavelength LED sensing (Red & IR)
  • Real-time SpO₂ calculation
  • Heart rate detection
  • Signal processing for noise rejection
  • Compact wearable design

Technologies Used

Embedded C Signal Processing Hardware Design Biomedical

Project Details

This biomedical device project involved designing both the hardware (LED/photodiode sensing circuits) and the embedded firmware for signal acquisition and processing. The device can accurately measure blood oxygen saturation and heart rate in real-time.
