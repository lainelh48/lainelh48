CANX-2 Satellite Link Simulator

RF Communications | MATLAB | SDR | Signal Processing

Overview

Developed a comprehensive MATLAB-based satellite communication simulator for the CANX-2 mission. The simulator models complex RF propagation phenomena including Doppler shift effects, path loss calculations, satellite tumbling attenuation, and includes hardware playback capabilities for Software Defined Radio (SDR) testing.

Key Features

  • Dynamic Doppler frequency modeling
  • Path loss and atmospheric attenuation calculations
  • Tumbling satellite attitude effects
  • Real-time SDR hardware integration

Technologies Used

MATLAB Signal Processing SDR RF Systems

Project Details

This project involved developing simulation algorithms that accurately model satellite communication channels with realistic propagation effects. The system is designed to test communication protocols and signal processing algorithms before deployment.
