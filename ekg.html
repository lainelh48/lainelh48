Designed a multi-channel ECG acquisition and analysis system for cardiac signal monitoring. Implemented signal conditioning, filtering, and real-time analysis algorithms for heart rate and arrhythmia detection with a user-friendly interface.

Key Features:

  • Multi-channel ECG acquisition
  • Analog signal conditioning and filtering
  • Heart rate variability (HRV) analysis
  • Arrhythmia detection algorithms
  • Real-time visualization and logging
