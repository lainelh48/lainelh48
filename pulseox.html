Developed a non-invasive pulse oximetry device for real-time blood oxygen saturation (SpO₂) measurement. The system utilizes red and infrared LED sensors with advanced signal processing algorithms to extract physiological parameters from photoplethysmographic signals.

Key Features:

  • Dual-wavelength LED sensing (Red & IR)
  • Real-time SpO₂ calculation
  • Heart rate detection
  • Signal processing for noise rejection
  • Compact wearable design
